Esta é a segunda atividade sobre Lógica de Programação do curso **Técnico em Desenvolvimento de Sistemas**. Desta vez o foco foi dado aos pseudocódigoso, trabalhando com o programa VisuAlg

O desafio consistiu na criação de sistemas pelo VisuAlg para resolução de desafios matemáticos com processos estruturados, utilizando **Pseudocódigo** e **Fluxogramas**.

## ⚠️ Regras Importantes 

> **Atenção aos Detalhes:** Lógica é precisão, se o passo a passo conter erros, o algoritmo falha. Não esquecer de separar os tipos de dados e que bons códigos possuem identação e comentários
> **Estrutura:** Agora as tarefas possuem estruturas mais complexas e detalhadas, como: Entrada; Display; Entrada manual; Processamento; Saída.

---

## 📂 Estrutura do Repositório

* `fluxogramas`: Representações visuais do fluxo de cada tarefa.
* `pseudocodigos`: Arquivos contendo a lógica escrita de forma estruturada.
